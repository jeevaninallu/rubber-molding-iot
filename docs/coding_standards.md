Python Coding Standards

Follow PEP8.

Use snake_case for variables and functions.

Use PascalCase for classes.

File names: lowercase_with_underscores.py.

Indentation: 4 spaces.

Comments:

Use # above logic blocks.

Write docstrings using triple quotes.

Do not use unused imports.

Keep functions short and modular.

JavaScript / React Standards

Follow Airbnb JavaScript Style Guide.

Variables: camelCase.

Components: PascalCase.

One React component per file.

Use functional components + hooks.

Avoid inline CSS; use component-level styles.

General Naming Rules

Folder names: lowercase

Constants: UPPERCASE

Environment variables: UPPER_SNAKE_CASE

Branch names: feature/your-feature-name
